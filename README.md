# RCProxy
A lightweight inapp HTTP requests logger for your iOS and tvOS apps.

+ Simply log requests in your app with 2 lines of code.
+ Magnified JSON view.
+ cURL sharing.
+ JSON file sharing.
+ Copy a single row on long press.

## Installation
Use SPM.
